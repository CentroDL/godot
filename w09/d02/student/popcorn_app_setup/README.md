# Create a Rails Application

Part 1: Build up the app as we know it.

1. Use the setup to get the database set up.
  - name the database `popcorn_app_development`
1. Create a new Rails project
  - use the flag to setup up a PostgreSQL database connection
  - use the two flags `--skip-turbolinks --skip-test-unit`
1. Add necessary gems to the Gemfile (including, but not limited to, Pry).
1. Create a `User` model and test it in the `rails console`.
1. Add the correct routes using the Rails DSL in `config/routes.rb`.
